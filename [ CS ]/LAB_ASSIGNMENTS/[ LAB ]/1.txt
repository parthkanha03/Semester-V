### **Wireshark Setup & Tasks (Short Notes)**

#### **1. Install Wireshark (Windows)**

1. Visit: [Wireshark Download](https://www.wireshark.org/download.html)
2. Download the installer.
3. Install **WinPcap** (for packet capture) and **TShark** (command-line tool).
4. Launch Wireshark.

#### **2. Capture ICMP Packets**

1. **Open Wireshark**.
2. Choose **Ethernet/Wireless/Loopback interface**.
3. Use filter: `icmp`.
4. Start capture (click shark fin).
5. For Loopback: Ping localhost (`ping 127.0.0.1`).

#### **3. Analyze ICMP Data for OS Differences**

1. **TTL values** to identify OS:

   * Windows: TTL = 128
   * Linux: TTL = 64
   * macOS: TTL = 255

#### **4. Capture & Analyze DNS Packets**

1. Filter: `dns`.
2. Perform DNS query: `nslookup www.google.com`.
3. **Packet Layers**:

   * Ethernet -> IP -> UDP -> DNS

#### **5. Capture Based on IP Address**

1. Filter by Source IP: `ip.src == 192.168.x.x`
2. Filter by Destination IP: `ip.dst == 192.168.x.x`
3. Or: `ip.addr == 192.168.x.x`

#### **6. Export Packet Info to Text**

1. After capture, stop.
2. **File > Export Packet Dissections > As Plain Text**.
3. Save to your desired location.

#### **7. Follow TCP Stream**

1. Right-click on a TCP packet.
2. Select **Follow > TCP Stream**.
3. View the full conversation.

#### **8. Useful Filters**

* **Capture Filters** (Before capture):

  * `icmp`, `port 53`, `tcp port 80`, `host 192.168.x.x`
* **Display Filters** (After capture):

  * `ip.src == 192.168.x.x`, `tcp.port == 443`, `http`, `frame.len > 1000`
